## 安卓基础

#### Linux下的开发测试环境

1. java sdk
2. download the SDK Tools（最好使用 android-sdk_r24.0.2-linux.tgz ，24版本支持命令行进行所有操作，也支持界面操作）
3. ./tools/bin/sdkmanager --list
4. install other tools
