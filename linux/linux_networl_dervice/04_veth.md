# veth

## veth设备的特点

* veth 和其它的网络设备都一样，一端连接的是内核协议栈。
* veth 设备是成对出现的，另一端两个设备彼此相连
* 一个设备收到协议栈的数据发送请求后，会将数据发送到另一个设备上去。

veth设备的特点如下图所示：

```

|192.168.1.11  ↑               ↑               ↑                 |
|              |               +---------------+                 |
|              |         192.168.2.11     192.168.2.1            |
+--------------|-------------------------------------------------+
               ↓
         Physical Network

```
